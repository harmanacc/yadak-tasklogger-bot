# Services Layer

This folder contains business logic services.

## Files

- `azure-devops.ts` - Azure DevOps API integration for daily reports
- `scheduler.ts` - Job scheduler (polls every 60 seconds)
- `group-service.ts` - Group management (approve/reject)
- `user-service.ts` - User management
- `work-service.ts` - Work session handling (start/finish)
